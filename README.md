# The-World-Bank
Deciphering Global Debt: Insights from The World Bank's Data in SQL

Countries, much like individuals, accrue debt to bolster their economies and support infrastructural projects. This project delves deep into The World Bank's data, shedding light on international debt trends spanning from 1970 to 2015. By scrutinizing the debt profiles of developing countries, I aim to answer pivotal questions about the sheer magnitude of debt, the nations grappling with the highest liabilities, and the recurring debt indicators that dominate the landscape.
